Enter the URL to disable this extension (enable referrer) like ^www.google.com$. Setting that value, Chrome will send referrer from www.google.com.

NOTICE: Some sites check referrer, and you may not see these sites correctly, e.g. github.com/new. Add these sites as excluded URLs to bypass troubles.

You need to set the URL as regular expression. Here is the example of google.com.

perfect matches www.google.com/foo ^www.google.com/foo$
matches any path www.google.com
www.google.com/foo
www.google.com/foo/bar		 ^www.google.com/*
matches any host google.com
www.google.com
drive.google.com		 google.com$
matches any path and host google.com
google.com/foo
www.google.com
drive.google.com/foo/bar		 google.com/*

You don't need to mind http(s), a trailing slash, and escaping regular expression special characters.
